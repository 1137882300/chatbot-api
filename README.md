# 《ChatGPT AI 问答助手》开源项目

## 项目介绍

1. 技术栈：Java、SpringBoot、爬虫、ChatGPT、job、Docker
2. OpenAi Keys 申请：[https://beta.openai.com/account/api-keys](https://beta.openai.com/account/api-keys) - 用于处理扫码知识星球问题进行调用获取答案。
